# Python_PBL
model training -Clinical Risk Assesment for Type-2 Diabetes
#Over view
This project uses Machine Learning to predict the risk of Type-2 Diabetes based on health parameters like Glucose, BMI, Age, and Blood Pressure. Early detection helps prevent serious complications.

# Objective
- Predict whether a person has diabetes or not  
- Identify important health features  
- Focus on high recall to avoid missing diabetic cases

# Pipeline
Data Generation → Label Encoding → Feature Selection → StandardScaler → Logistic Regression → Evaluation

# Features Used
- Glucose   
- BMI  
- Age  
- Blood Pressure  
- Diabetes Pedigree Function  
- Skin Thickness

 # Model
- Logistic Regression (Binary Classification)

# Results
- Accuracy: ~97%  
- Recall: ~95%  

The model performs well in identifying diabetic patients.
